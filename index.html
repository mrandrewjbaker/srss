srss: sunrise-sunset detection library



srss.time.isDay

Returns true if the user's time is day time or false if it is night time.

srss.time.isNight

Returns true if the user's time is night time or false if it is day time.

Running functions at night and day

You have the ability to run your own functions based on day or night. You can insert your own function calls inside srss.dayFunctions or srss.nightFunctions.
